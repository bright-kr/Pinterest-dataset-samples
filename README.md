# Pinterest 데이터셋 샘플

<h2>1001개의 Pinterest 프로필 및 게시물로 구성된 샘플 データセット</h2>

![Pinterest dataset header](https://github.com/bright-kr/Pinterest-dataset-samples/blob/main/pinterest-datasets.PNG)

1000개 이상의 프로필 및 게시물로 구성된 Pinterest データセット 샘플입니다. データセット는 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>データセット에 포함된 일부 データ 포인트는 다음과 같습니다:</h2>

* ```url```: Pinterest 게시물의 URL
* ```post_id```: 각 게시물의 고유 식별자
* ```title```: 게시물의 제목
* ```content```: 게시물의 텍스트 콘텐츠
* ```date_posted```: 게시물이 게시된 날짜
* ```user_name```: 콘텐츠를 게시한 사용자의 사용자명
* ```user_url```: 사용자의 Pinterest 프로필 URL
* ```user_id```: 사용자의 고유 식별자
* ```followers```: 사용자의 팔로워 수
* ```likes```: 게시물이 받은 좋아요 수
* ```categories```: 게시물과 관련된 카테고리 또는 태그
* ```source```: 게시물의 출처
* ```attached_files```: 게시물에 첨부된 파일
* ```image_video_url```: 게시물에 첨부된 이미지 또는 비디오의 URL
* ```video_length```: 게시물에 첨부된 비디오의 길이
* ```hashtags```: 게시물에 사용된 해시태그
* ```post_type```: 게시물의 유형
* ```comments_num```: 게시물의 댓글 수
* ```comments```: 게시물 댓글의 텍스트 콘텐츠
* ```discovery_input```: Discovery input 값

그 외에도 매우 많습니다.

이는 "Pinterest Profiles and Posts (public data)"
データセット에서 파생된 샘플 하위 집합이며, 해당 データセット에는 <b>186,000,000개 이상의 프로필 및 게시물</b>이 포함되어 있습니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet. 선택적으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 전달 유형 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 주기: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>.

추출된 データ 포인트에 추가로 제공 가능한 データ enrichment: <b>요청에 따라 제공됩니다.</b>

<h2>Pinterest データセット의 활용 사례는 무엇입니까?</h2>

<h3>1. 소비자 트렌드</h3>
기업은 Pinterest データセット를 활용하여 실시간으로 부상하는 소비자 트렌드를 식별하고 이에 대응합니다. 인기 핀과 보드를 분석함으로써 기업은 변화하는 고객 관심사와 선호도에 대한 인사이트를 확보하고, 최신 트렌드에 부합하도록 제품 제공 및 마케팅 전략을 조정할 수 있습니다.

<h3>2. 시장 조사</h3>
브랜드는 Pinterest データセット를 분석하여 사용자 상호작용과 선호도를 이해함으로써 소셜 미디어 존재감을 최적화합니다. 영향력 있는 계정을 식별하여 인플루언서 마케팅 캠페인을 개선할 수 있으며, 이를 통해 타겟 오디언스 내에서 더 넓은 도달 범위와 더 높은 신뢰도를 확보할 수 있습니다.

<h3>3. 제품 개발</h3>
기업은 Pinterest データ를 활용하여 제품 개발과 혁신을 촉진합니다. 사용자 활동을 모니터링하면 시장의 공백과 충족되지 않은 소비자 니즈가 드러나며, 이를 통해 기업은 참여도와 관심이 높은 오디언스를 대상으로 새로운 제품을 만들 수 있습니다.
